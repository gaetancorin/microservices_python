/////////////INSTALLATION ENVIRONNEMENT VIRTUEL///////////////

avoir python3 --version
    pip3 --version
    virtualenv --version
    (virtualenv permet de créer un environnement virtuel avec des installations de paquets python indépendant du pc host.)

créer l'environnement virtuel= 
• "virtualenv -p python3 docker-flask"

ouvrir le powershell administrateur et enlever la restriction de fichier .ps1=
• "Set-ExecutionPolicy RemoteSigned" 
https://docs.microsoft.com/fr-fr/powershell/module/microsoft.powershell.core/about/about_execution_policies?view=powershell-7.2

activer l'environement virtuel=
• "docker-flask/Scripts/activate"(windows)
• "source docker-flask/bin/activate"(linux) 

ouvrir le powershell admin et remettre la restriction de fichier .ps1 = 
• "Set-ExecutionPolicy Restricted"

---NOUS SOMMES DESORMAIS DANS LE VIRTUALENVIRONMENT(ou venv)---

Flask est un microframework idéal pour des petites API(microservices)
• pip install flask

Freeze est un paquet native de python qui permet de voir les dépendances installer
voir les dépendances installer dans le venv avec freeze = 
• "pip freeze"
crée/Met à jour le fichier requirements.txt en comparant avec les dépendances enregistrer de freeze =
• "pip freeze > requirements.txt"
