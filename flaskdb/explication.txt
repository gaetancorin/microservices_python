• virtualenv -p python3 docker-flask    créer venv
• docker-flask/Scripts/activate         activer venv
• pip install flask                     installer flask

• pip install flask_pymongo         installe pymongo qui connecte a mongoDb
• pip install dnspython        installe connection dns indispensable pour pymongo
• pip freeze > requirements.txt

• python app.py
